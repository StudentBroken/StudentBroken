I'm a 17-year-old developer and maker from Quebec, Canada. I've been building hardware and software systems since I was 12, documenting my journey and projects in a digital archive I call *Habanero Cheeseburger* (`imnotabot.dev`). 

Whether it's deploying cross-platform apps, reverse-engineering smart glasses, programming custom ESP32 IoT networks, or designing 3D-printed FPV drones—I love turning complex, multi-component ideas into functional realities.

### 🚀 What I've Been Working On
- **RepGate** : A cross-platform parental control app that allows kids to earn screen time by completing verified exercise sessions.
- **Solos HUD Redesign** : Reverse-engineered Solos smart glasses to build custom applications, combined with a ground-up hardware redesign and 3D-printed housing.
- **MBS Grade Tracker** : A collaborative grade-tracking web platform currently used and maintained for Quebec students.
- **Custom E-Bikes & Hardware** : Currently on my 6th iteration of a custom electric bike, supported by DIY ESP32-based smart chargers and 48V USB-C PD adapters.
- **Drone & RC Engineering** : Built custom 2-axis head-tracking DJI O3 setups, 3D-printed FPV drones (including ones that spray water and shoot gel blasters!), and a fully 3D-printed RC F1 car.
- **V Plotter: V-Bot** : A portable, crab-inspired dual-stepper V-Plotter with a custom web interface.

### 🛠️ My Toolbox
- **Software**: Full-Stack Web & Mobile App Development, C/C++ (Embedded), Python
- **Hardware & IoT**: ESP32 / Arduino, Raspberry Pi, Klipper, MQTT, BLE, Deep Sleep Optimization
- **Making**: 3D Modeling (Onshape), 3D Printing (TPU, PEI, custom mechanics), PCB Soldering, Hardware Reverse Engineering
